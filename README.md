timelog
=======
Timelog is a simple file reader for timelogs, similar to [ledger-cli](https://www.ledger-cli.org/).

&nbsp;

Sample timlog files
-------------------

``` markdown
### My project

2020-01-01
    10:00-12:32 meeting with client
    12:32-13:00 prep. meeting notes
    2.15h project work

2020-01-02"
    15m

```
